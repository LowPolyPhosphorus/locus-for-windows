# Locus

A macOS focus app that blocks distracting apps and websites during study
sessions. Pulls upcoming work from your Notion planner or any calendar
feed (Google Calendar, Apple Calendar, Schoology, etc.) so you can start
a session straight from an assignment.

Website: **https://locusfoucsapp.netlify.app**

## Install

1. Download the latest **Locus.dmg** from the
   [Releases page](../../releases/latest).
2. Open the DMG and drag **Locus** into your **Applications** folder.
3. First launch: right-click **Locus** → **Open** → **Open** (the app
   isn't notarized yet, so macOS warns once).

That's it. Open Locus, optionally connect Notion or paste a calendar
URL under **Connectors**, then start a session.
